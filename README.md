# 🛡️ Port Scanner em Python

 Sobre o projeto

Esse é um projeto que eu desenvolvi enquanto estudo cibersegurança, com foco em pentest.
A ideia foi criar um scanner de portas simples em Python, inspirado em ferramentas como o Nmap, pra entender melhor como funciona a comunicação em rede e a identificação de serviços ativos.

Foi um projeto simples, mas que me ajudou muito a evoluir na prática.

---

 O que ele faz

* Escaneia portas de um alvo (site ou IP)
* Mostra quais portas estão abertas
* Tenta identificar o serviço rodando em cada porta

---

 Tecnologias

* Python 3
* Biblioteca socket

---

 Como usar

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/port-scanner.git
```

2. Entre na pasta:

```bash
cd port-scanner
```

3. Execute:

```bash
python scanner.py
```

4. Digite o site ou IP quando pedir (ex: google.com)

---

 Exemplo

```bash
Escaneando alvo: google.com

Porta 80 ABERTA - Serviço: http  
Porta 443 ABERTA - Serviço: https  
```
